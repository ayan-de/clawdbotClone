# Orbit CLI Releases

This directory contains platform-specific binaries for the Orbit CLI tool.

## Binary Naming Convention

- `orbit-linux-x64` - Linux x86_64
- `orbit-linux-arm64` - Linux ARM64
- `orbit-macos-x64` - macOS Intel
- `orbit-macos-arm64` - macOS Apple Silicon

## Installation

Install the latest version using:

```bash
curl -fsSL https://orbit.ayande.xyz/install.sh | bash
```

## Verification

Verify your download with the checksums file:

```bash
sha256sum -c checksums.txt
```

## Version History

Check available versions at: https://api.orbit.ayande.xyz/releases
